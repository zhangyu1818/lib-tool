# lib-tool

Zero configuration, easy way to package a library.

**Features**

- Zero configuration.
- Support JavaScript, TypeScript, React.
- Support both ESM and CJS.

# Installation

**npm**

```sh
npm install -D lib-tool
```

**yarn**

```sh
yarn add lib-tool --dev
```

### Setup

**package.json**

```json
{
  "scripts": {
    "build": "lib-tool"
  }
}
```
